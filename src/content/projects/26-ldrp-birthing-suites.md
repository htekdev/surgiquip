## Project Overview

This project addressed persistent problems with an existing door-mounted birthing light system — dust accumulation and mechanical jamming that disrupted clinical workflows.

## Scope of Work

### LDRP Suites (26 Rooms)

- **Skytron Mini Argos Ceiling Birthing Lights** — replacing the problematic door-mounted system
- **Wand Control Systems** — wireless control for precise light positioning
- **Ceiling-Mount Hardware** — structural mounting for each suite

## The Problem Solved

The previous door-mounted lights suffered from:
- **Dust accumulation** in the track mechanism
- **Mechanical jamming** — lights would stick or resist movement
- **Staff frustration** — unreliable positioning during deliveries

## The Solution

Ceiling-mounted lights eliminate the track mechanism entirely. Wand control allows precise, effortless positioning from bedside.
